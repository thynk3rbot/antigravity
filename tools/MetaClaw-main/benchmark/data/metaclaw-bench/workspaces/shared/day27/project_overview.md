# Project Orion — Project Overview

Last Updated: April 10, 2026

## Project Description

Project Orion is a B2B SaaS project management platform designed for distributed teams. The platform provides task management, workflow automation, real-time collaboration, and analytics capabilities.

## Technology Stack

- **Backend**: Python 3.11, FastAPI, PostgreSQL
- **Frontend**: React 18, TypeScript, TailwindCSS
- **Infrastructure**: AWS (ECS, RDS, S3), Docker, Terraform
- **CI/CD**: GitHub Actions, ArgoCD

## Team Structure

- **Engineering**: 8 developers (3 backend, 3 frontend, 2 full-stack)
- **Product**: 2 product managers
- **Design**: 1 UX designer
- **QA**: 2 QA engineers
