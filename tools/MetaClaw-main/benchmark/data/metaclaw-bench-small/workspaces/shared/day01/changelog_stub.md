# Orion API Changelog

## Version 2.3.0

**Released:** March 19th, 2026

### Added
- New /tasks endpoint for task management
- Support for task dependencies and subtasks
- Enhanced filtering options for /projects endpoint

### Changed
- Improved error messages for validation failures
- Updated rate limiting policies

### Fixed
- Fixed pagination issue in /projects endpoint
- Corrected timezone handling in date range queries

---

## Version 2.2.0

**Released:** 2026/02/28

- Batch operations support for project updates
- New webhook endpoints for real-time notifications
- Support for custom fields in project metadata

### Changed
- Optimized database queries for large datasets
- Improved API response times

### Fixed
- Fixed issue with concurrent project modifications
- Resolved memory leak in long-running connections
