Everything You Need
in One Firmware

Six major subsystems, one unified codebase — no glue code required.

📡

LoRa Mesh

RadioLib SX1262 driver with hardware abstraction. AES-128 payload encryption, configurable spreading factor and bandwidth, and real-time RSSI/SNR telemetry. Nodes relay packets across multi-hop topologies automatically.

🔗

ESP-NOW

Peer-to-peer Wi-Fi without an access point. MAC-based peer registry with up to 10 peers persisted in NVS. Incoming messages queue to 8 slots and are dispatched through the same CommandManager routing layer.

📶

MQTT Integration

Publish device telemetry and receive commands from any MQTT broker. Works alongside the local mesh — cloud visibility without replacing local-first operation. Supports TLS brokers via PubSubClient.

🗓

Task Scheduler

Priority-queue task engine with up to 5 dynamic slots. Schedule TOGGLE or PULSE actions on any GPIO pin or relay at arbitrary intervals. Tasks survive reboots via NVS persistence.

📱

BLE Terminal

Nordic UART Service (NUS) GATT server exposes a raw serial terminal over Bluetooth. Control nodes directly from a mobile app or desktop BLE client. Devices advertise as GW-XXXX for easy discovery.

🔄

OTA Updates

Build once, update everywhere. Firmware pushed over Wi-Fi via mDNS hostnames — no hardcoded IPs, no physical access required. Supports parallel fleet updates with a single pio run -e ota_master.
