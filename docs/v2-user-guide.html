🚀 Magic V2 User Guide

The definitive manual for the v2.0 "Sovereign" firmware release.

⚡ Getting Started

V2 introduces Auto-Discovery and Hardware Agnostic Plugins. To begin:

  • Connect your Heltec V3/V4 to USB.
  • Open the Fleet Admin dashboard.
  • Watch as your device automatically appears in the "Assets" sidebar via mDNS.

🔗 Mesh Architecture

Tier 1: LoRa (10km)

The backbone. Uses SX1262 (V3/V4) or SX1276 (V2) for ultra-long-range command delivery.

Encrypted

Tier 2: ESP-NOW (200m)

High-speed, low-power peer-to-peer. Perfect for high-frequency telemetry swarm updates.

Fast

Tier 3: WiFi (LAN)

High-bandwidth local access. Required for Webapp, REST API, and OTA updates.

Web UI

🛠️ Hardware Plugin System

V2 decouples logical peripherals from physical pins. You can drive any device on any board.

Plugin Description Config command
DHT Sensor Temp/Humidity monitoring SETDHT [PIN] [TYPE]
RelayHAL Power control (Native or MCP23017) RELAY [0-7] [ON/OFF]
RadioHAL LoRa tuning (Freq, BW, Preamble) SETRADIO [FREQ] [BW]

🚨 Troubleshooting

"Node Offline" in Dashboard

If a node appears greyed out:

  1. Verify the device has power (OLED should show heartbeat).
  2. Ensure the Zeroconf/mDNS service is running on your PC.
  3. Run a "Subnet Scan" from the Hardware tab to force-refresh the registry.
